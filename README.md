# Portafolio Web - Sebastián Ayenao 

Se trata de un portafolio digital dinámico creado para destacar mi experiencia y habilidades profesionales. Integra un panel de control que permite administrar, editar y actualizar todo el contenido público de forma ágil y centralizada.

Enlace del proyecto en produccion: https://teclab.uct.cl/~sayenao2025/index.php

## Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **Backend:** PHP
* **Base de Datos:** MySQL
* **Arquitectura:** API REST (consumo mediante peticiones asíncronas AJAX/Fetch API)

## Instalación y Configuración
1. Clonar o descargar este repositorio en la carpeta `htdocs` de tu servidor local (XAMPP/WAMP).
2. Crear una base de datos en phpMyAdmin (por ejemplo: `portafolio_bd`).
3. Importar el archivo `database.sql` incluido en este repositorio.
4. (Opcional) Ajustar las credenciales en el archivo `conexion.php` si tu usuario root tiene contraseña.

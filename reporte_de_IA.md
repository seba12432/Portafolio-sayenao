# Reporte de Uso de Inteligencia Artificial 


##  1. Contexto de Uso
En el marco del desarrollo del proyecto final "Portafolio Web", se utilizó Inteligencia Artificial (específicamente modelos de lenguaje LLM) como herramienta de asistencia técnica, depuración de código y estructuración de arquitectura web. El uso se enfocó en optimizar los tiempos de resolución de problemas, aplicar buenas prácticas y comprender la integración profunda entre **PHP, MySQL y JavaScript**.

##  2. Áreas de Aplicación
* **Depuración de errores (Debugging):** Asistencia para identificar problemas en la interfaz y resolver bloqueos de validación estricta en formularios HTML5.
* **Explicación de Conceptos:** Comprensión detallada de cómo estructurar interacciones asíncronas (AJAX) con la base de datos.
* **Generación de Contenido Base:** Creación de textos descriptivos profesionales y variaciones de redacción para las secciones de biografía y descripciones de proyectos específicos como "Kaeli".
* **Diseño de Interfaz:** Sugerencias de integración de clases de Bootstrap 5 para renderizar iconos, colores específicos y barras de progreso dinámicas.
* **Control de Versiones:** Asistencia paso a paso para el flujo de trabajo en Git, separación de commits y la gestión del repositorio mediante GitHub Desktop.

##  3. Conclusión Tecnológica
La integración de herramientas de IA permitió no solo la finalización exitosa del código fuente del portafolio, sino también un aprendizaje significativo sobre el manejo del DOM, la seguridad en peticiones HTTP y el control de versiones, cumpliendo así con los estándares de calidad exigidos para el perfil.
